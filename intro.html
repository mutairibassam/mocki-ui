Mocki

Mocki is a HTTP benchmarking and data generation tool. No scripts or manual data generation anymore!

generate traffic simulating users or applications that access a site application or api and see how much traffic that application can handle before it becomes slower or inaccessible

**more bandwidth does not improve load time**

Appendix

  • Pipelining: The number of pipelined requests to use. default: 1. Note: backend pipelining support is required, otherwise will be defaulted to 1.
  • Number of Connection: The number of concurrent connections to use. default: 10.
  • Max connection: The max number of requests to make per connection to the server. default: 10
  • Duration: The number of seconds to run the autocannon. default: 10.
